    - `validateGstin(string $v): array` — regex `/^[0-9]{2}[A-Z]{5}[0-9]{4}[A-Z]{1}[1-9A-Z]{1}Z[0-9A-Z]{1}$/` (15 chars)

    - `validateRate(float $v): array` — range 0.00–100.00
    - `validateDaysInt(int $v): array` — range 1–365
    - `validateSlug(string $v): array` — regex `/^[a-z0-9-]{3,100}$/`
    - `validateLabel(string $v): array` — length 3–500

    Use the `['ok' => bool, 'error' => string|null]` return shape consistently. The endpoint code can then do `if (!$r['ok']) jsonResponse(false, null, $r['error'], 400)`.

    - `declare(strict_types=1);` at the top
    - The function creates `quotation_counters` table on first call (idempotent CREATE TABLE IF NOT EXISTS)
