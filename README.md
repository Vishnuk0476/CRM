# Panya Global

A modern logistics and consignment tracking website built with React, Vite, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Consignment tracking
- Quote requests
- Service inquiries
- Admin dashboard
- Branch locations
- Client showcase
- Online payments

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router
- Radix UI Components

## Project Structure

```
src/
├── components/     # React components
├── pages/          # Page components
├── hooks/          # Custom hooks
├── lib/            # Utilities
├── services/       # API services
├── data/           # Static data
└── assets/         # Images and media
```
